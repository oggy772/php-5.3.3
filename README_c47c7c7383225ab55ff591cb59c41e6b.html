php-5.3.3 ビルド

目的

php 5.3.3 のソースコードから json_encode, json_decode の API のライブラリをビルドする。

はじめに

PHP ビルドの方法は Build your own PHP on Windows を参照。

1. PHP 5.3.3 のソースコードをダウンロード

http://php.net/releases/　の一覧から 5.3.3 をダウンロード。 php-5.3.3 に解凍。

2. PHP ビルド用パッケージをダウンロード

php-sdk-binary-tools-20110915.zip をダウンロード。 php-5.3.3 以下に解凍（bin, script, share）。
